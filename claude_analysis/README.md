## Quick Start

### Running Benchmarks
```bash
# Basic benchmark
python3 benchmark_claude.py

# With custom parameters
python3 benchmark_claude.py --iterations 100 --model claude-3-opus

# Save results
python3 benchmark_claude.py --output benchmarks/results_$(date +%Y%m%d).json
```
