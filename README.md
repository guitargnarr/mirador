## Overview

Mirador is a sophisticated multi-agent AI orchestration system that implements dynamic agent chains for complex reasoning tasks. Built on Ollama for local LLM execution, it demonstrates production-scale agent coordination, context management, and intelligent routing.

**Key Achievement**: 141,000+ lines of code implementing enterprise-grade agent orchestration patterns.

## Core Features

### Multi-Agent Orchestration
- **80+ Specialized Agents**: Domain experts, strategy architects, context providers
- **Dynamic Chain Composition**: Intelligent routing based on query analysis
- **Context Accumulation**: Information flows and enriches through agent pipelines
- **Parallel Agent Execution**: Concurrent processing for efficiency

### Intelligent Routing
Smart query routing automatically selects optimal agent chain based on intent analysis.

### Performance & Scale
- **Local Execution**: All processing on-device via Ollama
- **No External APIs**: Privacy-first architecture
- **Apple Silicon Optimized**: Leverages M-series GPU acceleration
- **Production Ready**: Comprehensive test suite, error handling, monitoring

## Technical Stack

- **Language**: Python 3.9+, Bash
- **LLM Backend**: Ollama (local inference)
- **Models**: Llama, Gemma, Qwen, Phi, Command-R, DeepSeek
- **Storage**: SQLite (session persistence)
- **Architecture**: Multi-agent pipeline with context accumulation
